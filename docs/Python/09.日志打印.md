简单配置如下：
```python
import logging

logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s [%(levelname)-5.5s]:%(message)s",
    handlers=[
        logging.FileHandler("all.log", 'w', 'utf-8'),
        logging.StreamHandler()
    ]
)

logging.info('hello world')
```
