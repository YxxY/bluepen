## 安装
```shell
npm install mockjs
```

## 模拟数据
`Mock.mock(template)`
Mock 需要的目标数据，该部分需使用指定的规则

## 拦截请求
`Mock.mock([rurl,] [rtype,] template|function( options ))`  
实现拦截 `AJAX`请求，并返回指定数据结构的功能。
